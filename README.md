## How To Run
- Pastikan setiap .env sudah benar
- Pastikan Docker sudah terinstall
- Buka terminal di direktori project ini
  ```bash
  ./deploy.sh          # Start
  ./deploy.sh logs     # Check logs
  ./deploy.sh stop     # Stop
  ./deploy.sh restart  # Restart
  ```
